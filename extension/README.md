# PCAPViewer

pcap/pcapng file analyzer


## protocol

- Ethernet II
- SLL
- IPv4
- IPv6
- ARP
- TCP
- UDP
- ICMP
- ICMPv6
- IGMP
- DNS
- DHCP
- TLS
- HTTP

support *.pcap *.pcapng files
