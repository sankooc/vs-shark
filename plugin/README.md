# pcapviewer

**pcapviewer** is an open-source tool designed for analyzing **PCAP / PCAPNG (Packet Capture)** files.  
It provides a lightweight and efficient way to inspect network packets across multiple protocol layers ‚Äî from Ethernet to Application ‚Äî without the heavy overhead of full-scale tools like Wireshark.

---

### üß≠ Overview

pcapviewer helps developers and network engineers quickly visualize and analyze captured network traffic.

#### Supported Layers

- **File Formats**: PCAP, PCAPNG  
- **Link Layer**: Ethernet II, PPPoE, IEEE 1905.1a, IEEE 802.11, SSL  
- **Network Layer**: IPv4, IPv6, ARP, RARP, ICMP, IGMP  
- **Transport Layer**: TCP, UDP  
- **Application Layer**: TLS, HTTP, DNS
